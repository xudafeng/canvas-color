## Usage

```javascript
new CanvasColorPicker({
  image: 'https://avatars2.githubusercontent.com/u/9263023?v=4&s=400',
  size: 40,
  radius: 1.2,
  onSuccess: function(e) {
    // red green blue
    console.log(e);
  }
});
```
